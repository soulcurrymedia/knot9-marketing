Explore the Syllabus

Learning at Code Gear Labs is a completely different experience from learning at other educational institutions. It encompasses a diversity of learning experiences that help you absorb knowledge, practice your skills, and get connected to the industry network.

Phase 1: Get Started

  1. System Setup - Ubuntu Installation
  2. Terminal basic commands
  3. HTML / CSS foundations
  4. Javascript foundations
  5. Ruby foundations
  6. Version Control
  7. Git basic commands
  8. GITHUB/GITLAB
  9. General knowledge about tech workflows in a startup
Prepwork enables our students to understand basic notions beforehand, through a clear introductory course. It facilitates the learning process for when we cover these concepts in detail during the program.

Phase 2: Programming basics

  1. Install Ruby and a Version Management tool
  2. Core concepts of programming
  3. Algorithms & data structures (Array, Hash)
  4. Text pattern detection with Regular Expressions
  5. Ruby doc for built-in classes
  6. Parse and store data from standardized files (JSON, CSV, XML)
  7. Request and parse data from RESTful JSON APIs

Phase 3: Software Architecture

  1. Core concepts of Object-Oriented programming
  2. Classes, objects and instance variables
  3. Instance and class methods, Inheritance
  4. Public vs. Private Interfaces
  5. Routing users' requests
  6. Simulating a database with CSV files
  7. In-depth understanding of MVC (Model View Controller)

Phase 4: Relational Database, SQL & ORM

  1. Database Schema Design & SQL
  2. One-to-many, many-to-many relations and join table
  3. CRUD in SQL (Create, Read, Update, Delete) Advanced JOIN queries Connecting your DB to ruby with an ORM (Object Relational Mapping)
  4. Active Record Basics: migrations & naming convention
  5. Active Record Advanced: associations & validation

Phase 5: Web Development

  1. Product Design, UX & UI principles
  2. HTML basics
  3. CSS (selectors, properties, stylesheets' organization)
  4. Atomic design principles
  5. JS and Jquery
  6. CSS techniques with flexbox & CSS grid CSS media queries for Adaptive Design
  7. Bootstrap basics & Responsive Grid System

Phase 6: Wrapping all together with Rails

  1. MVC in Rails: Routing, Controllers, Params & Views
  2. Models & CRUD in Rails
  3. Adding gems to your project
  4. ERB: Layouts, Partials and Helpers
  5. Nested Resources and Namespaces
  6. Refactoring an app with Services
  7. Defining a Rails API

Phase 7: Projects

Invent, design, code, deploy and pitch personal projects in teams. Projects Phase is the ultimate experience of the program: invent, pitch, design, code and deploy an original project in teams of 3 or 4 people during the two final weeks.

Q: Why is this course relevant today?

There’s never been a better time to start a career as a software developer. From startups to Fortune 500 companies, there is a growing demand for software developers who can creatively solve problems and implement robust, sustainable solutions. The quality software developer is always in demand.

Q: What practical skill sets can I expect to have after completing this course?

By the end of our Web Development Immersive, you will be able to:

  • Create responsive web pages for modern browsers using HTML, CSS and JavaScript.
  • Write secure full-stack applications and deploy them to cloud-based services like Heroku.
  • Consume web application programming interfaces (APIs) from third-party sites such as Twitter, Google, or Yelp.
  • Build a RESTful API using a technology like Sinatra, Rails
  • Build a richly interactive, front-end single-page application using a modern framework like Angular or library like React.
  • Collaborate as a team using Git and GitHub, widely accepted collaboration practices, and an agile development workflow.
